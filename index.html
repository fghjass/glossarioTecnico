Tags HTML

<div>

Elemento genérico usado para agrupar conteúdo.

<div>Conteúdo</div>

<img>

Insere imagens na página.

<img src="foto.jpg" alt="Descrição">

<a>

Cria um link para outra página ou local.

<a href="https://exemplo.com">Visitar</a>

<h1> a <h6>

Definem títulos e subtítulos de diferentes níveis.

<h1>Título</h1>

<p>

Representa um parágrafo de texto.

<p>Texto aqui.</p>

<ul> e <li>

Lista não ordenada com itens.

<ul>
    <li>Item</li>
</ul>

<ol> e <li>

Lista ordenada com itens numerados.

<ol>
    <li>Item</li>
</ol>

<table>

Cria tabelas de dados.

<table>...</table>

<form>

Cria formulários para entrada de dados.

<form>...</form>

<input>

Campo para entrada de dados no formulário.

<input type="text">

Propriedades CSS

color

Define a cor do texto.

p { color: red; }

background-color

Define a cor de fundo do elemento.

body { background-color: #f4f4f4; }

margin

Espaçamento externo do elemento.

div { margin: 10px; }

padding

Espaçamento interno do elemento.

div { padding: 10px; }

border

Adiciona bordas ao elemento.

p { border: 1px solid black; }

font-size

Tamanho da fonte.

h1 { font-size: 32px; }

text-align

Alinhamento do texto.

p { text-align: center; }

display

Controla o tipo de exibição do elemento.

div { display: flex; }

width

Largura do elemento.

img { width: 200px; }

height

Altura do elemento.

img { height: 150px; }
